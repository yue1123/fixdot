# fixdot
This is a command tools that fix english punctuation to chinese punctuation

# function
1. 命令行工具
   1. 支持指定文件
      1. 具体文件
      2. 文件夹
      3. 通配符文件 如: *.js *.ts
   2. 支持配置,具体配置还需要想一下
   3. 修复结果统计
      1. 变动了那些文件
      2. 浏览不正确的标点


# command
1. fixdot --preview -p
